# physics-constrained-vascular-gnn
A Physics-Constrained Graph Neural Network (PCGNN) designed for high-fidelity, real-time blood flow simulation in patient-specific liver networks. By integrating 1D Navier-Stokes equations into the MeshGraphNet architecture, this model achieves millisecond-scale inference while strictly maintaining mass and momentum conservation.
